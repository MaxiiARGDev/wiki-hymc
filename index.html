HygieneMC

La guía definitiva para el sistema de higiene más inmersivo de Minecraft. Mantén a tus jugadores limpios y con estilo.

Mecánicas Principales

Niveles de Higiene

El sistema rastrea un valor de 0 a 100. Disminuye con el tiempo y se acelera al correr, saltar o combatir.

Estados (Stages)

Desde "Limpio" hasta "Asqueroso". Los estados críticos emiten partículas de mal olor y sonidos ambientales.

Notificaciones

Alertas visuales vía ActionBar y Títulos para asegurar que el jugador sepa cuándo necesita un baño.

Comandos y Permisos

Comandos de Jugador

Comando Descripción Permiso
/hygiene Muestra tu nivel actual de higiene hygiene.use
/hygiene ui Abre el menú de configuración personal hygiene.use
/hygiene shop Abre la tienda global de higiene hygiene.use
/hygiene buysoap <tipo> [cant] Compra jabón específico desde cualquier lugar hygiene.use

Comandos de Administrador

Comando Descripción Permiso
/hygiene get <jugador> Consulta la higiene de otro jugador hygiene.admin
/hygiene set <jugador> <valor> Establece la higiene de un jugador (0-100) hygiene.admin
/hygiene add <jugador> <valor> Suma o resta higiene a un jugador hygiene.admin
/hygiene reload Recarga todos los archivos de configuración hygiene.admin
/hygiene givesoap <p> [c] [v] Entrega jabón directamente a un jugador hygiene.admin
/hygiene giveshower [jugador] Entrega el ítem de Ducha (Shower) hygiene.admin
/hygiene givetoilet [v] [j] Entrega el ítem de Inodoro (Toilet) hygiene.admin
/hygiene givelavatory [v] [j] Entrega el ítem de Lavabo (Lavatory) hygiene.admin
/hygiene givedecor <id> [j] Entrega un mueble decorativo específico hygiene.admin

Crafteos y Recetas

Puedes fabricar tu propio equipamiento de higiene usando materiales básicos en la mesa de crafteo.

Ducha de Hierro

Receta: 1 Botón de Piedra + 1 Lingote de Hierro.

(Configurable en shower.yml)

Inodoro Base

Receta: 1 Bloque de Cuarzo + 1 Cubo de Agua.

Resulta en el modelo básico de porcelana.

Jabón Casero

Receta: Mezcla tintes y materiales grasos (Configurable).

Crafteo sin forma en la mesa de trabajo.

Mobiliario e Ítems

Duchas Funcionales

Bloques interactivos con partículas de agua. Restauran la higiene al máximo en segundos.

Inodoros y Lavabos

Múltiples variantes de colores (Blanco, Azul, Rosa) para encajar en cualquier construcción.

Jabones Especiales

Ítems consumibles que limpian al instante. ¡Ideales para viajes largos!

Tienda y Economía

HygieneMC se integra totalmente con Vault, permitiendo un sistema económico fluido donde la higiene tiene un valor real en tu servidor. Los jugadores pueden abrir la tienda con /hygiene shop para comprar:

Jabones de Calidad

Compra jabones de diferentes calidades y eficacias directamente desde el menú de la tienda.

Mobiliario para Casas

Acceso a mobiliario funcional y decorativo para que los jugadores personalicen sus hogares.

Variantes Estéticas

Inodoros y lavabos de múltiples colores para encajar con cualquier estilo arquitectónico.

Instalación y Requisitos

  • Paper 1.21+: El plugin está optimizado para las últimas versiones de PaperMC.
  • Vault (Opcional): Necesario para habilitar el sistema de tiendas y precios.
  • PlaceholderAPI (Opcional): Para mostrar la higiene en Scoreboards o TabLists.
# Guía Rápida
1. Sube el hygienemc.jar a la carpeta /plugins
2. Reinicia o inicia el servidor
3. Personaliza los archivos YAML en /plugins/HygieneMC/

Preguntas Frecuentes

¿Cómo evito ensuciarme tan rápido?

Evita correr (sprint) y saltar excesivamente. Estar en reposo o caminando disminuye drásticamente el ritmo de suciedad.

¿El mal olor afecta a otros?

Visualmente sí. Otros jugadores verán las partículas de mal olor y escucharán sonidos si estás en el estado "Asqueroso".

¿Puedo desactivar el sistema en ciertos mundos?

Sí, puedes gestionar los permisos o usar plugins de gestión de mundos para limitar dónde actúa el sistema de higiene.
